# Login & User Management System

## Description
A simple Login and Registration system built using Python and SQLite.  
The project demonstrates backend development, database integration, and SQL constraints.

## Tech Stack
- Python
- SQLite
- Flask (basic usage)
- HTML

## Features
- User Registration
- User Login
- Session Management
- SQLite Database Integration
- Duplicate User Prevention using UNIQUE constraints

## Database
SQLite database with proper schema, UNIQUE and NOT NULL constraints.

## How to Run
1. Clone the repository
2. Install Flask  
   pip install flask
3. Run the application  
   python app.py
4. Open browser and visit  
   http://127.0.0.1:5000

## Note
This project focuses mainly on Python and SQLite.
